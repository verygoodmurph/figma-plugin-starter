# Figma Plugin Starter Template

Includes examples of core concepts:

- Plugin with a UI
- Communication between Code & UI
- Async functions for dynamic-pages (manifest)
- Updating the UI
- Light/Dark mode colors
- External fonts
- Notifications
- Logging to console
