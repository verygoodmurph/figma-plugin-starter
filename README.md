# Figma Plugin Starter Template

Demonstrates essential Figma plugin concepts:

- Custom UI integration
- Code and UI message passing
- Async operations with dynamic pages
- UI updates from plugin logic
- Light and dark mode support
- External font handling
- User notifications
- Console logging

### Recommended Stack for Non-Technical Users

- [Cursor](https://cursor.com/)
- [Figma](https://www.figma.com/)
- [Github](https://github.com/)
- [Github Desktop](https://desktop.github.com/download/)

### Cursor Commands

These are operational guides found in the `.cursor/commands` directory. Each markdown file describes step-by-step workflows for common tasks (e.g., setting up new features, creating READMEs). Use them as references for structured development and project onboarding.

### Cursor Rules

Rules and agent instructions (like those in `AGENTS.md`) document coding standards, architectural best practices, and automation guidelines for this repository. They help ensure code quality, maintainability, and consistent plugin development patterns.
